# LLM-Based Code Review Assistant

A web application that uses the Anthropic Claude API to analyze source code and provide automated feedback on bugs, security vulnerabilities, and code quality improvements.

**Course:** CSCI 334 — Communication for Computing Sciences  
**Institution:** St. Cloud State University  
**Author:** Pukar Puri

---

## What This Project Does

You paste source code into the web interface, select the programming language, and click Analyze. The app sends your code to the Claude API, which returns structured feedback in three categories: bugs, security vulnerabilities, and code quality suggestions. Results are displayed in color-coded panels.

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/PukarPuri17/llm-code-reviewer.git
cd llm-code-reviewer
```

### 2. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 3. Set up your API key
```bash
cp env.example .env
```
Open `.env` and replace `your_anthropic_api_key_here` with your real key from https://console.anthropic.com

### 4. Start the backend server
```bash
cd backend
python app.py
```
You should see: `Running on http://127.0.0.1:5000`

### 5. Open the frontend
Open `frontend/index.html` in your browser.

---

## Running the Tests
```bash
pytest tests/ -v
```
Tests mock the API so no key is needed and no API costs are incurred.

## Disclaimer
All feedback generated by this tool is produced by a probabilistic AI model. It may contain errors, miss real vulnerabilities, or suggest incorrect fixes. Never apply AI-generated suggestions without independent review and testing. This tool is a first-pass assistant, not a replacement for human code review.
